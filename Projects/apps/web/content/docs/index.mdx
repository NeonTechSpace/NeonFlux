NeonFlux is a Fluxer bot with a protected web dashboard.

The bot can run in two modes:

- Single instance: one configured community.
- Multi instance: one bot handles many installed communities.

The dashboard uses Fluxer OAuth. Users only see communities they can manage, and server code re-checks access before meaningful changes.

## Planned capabilities

- Autorole
- Import and export roles, channels, categories, emojis, and permissions
- Logging
- Message and embed posting
- Moderation
- Profile and about-me embed builder
- Reaction roles
- Suggestion votes
- Tickets
- VC generator
- Verification reaction roles
- XP and levels

## Routes

- `/docs/*` is public documentation.
- `/auth/*` handles Fluxer OAuth.
- `/dashboard/*` is protected dashboard UI.
